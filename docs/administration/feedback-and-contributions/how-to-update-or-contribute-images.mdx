Watch the video below for detailed instructions on how to update or add images to the documentation using our open source [GitHub repository](https://github.com/device42/device42-docs).

The process consists of the following steps:
* Navigate to your forked Device42 repository
* Create a new branch
* Upload the image to static/assets/images
* Change the file path in the markdown document
* Open a pull request
